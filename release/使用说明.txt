========================================
  MD5 + AES 加密解密工具 — 使用说明
========================================

一、软件简介
-----------
本工具提供以下功能：
  1. 文本加解密 — 使用 AES-256-GCM 算法对文本进行加密/解密
  2. 文件加解密 — 使用 AES-256-GCM 算法对任意文件进行加密/解密
  3. MD5 计算   — 计算文本或文件的 MD5 哈希值，支持 MD5 比对校验

二、运行环境
-----------
  - 操作系统：Windows 10/11（64位）
  - 无需安装 Python 或其他依赖，双击 .exe 即可运行

三、使用指南
-----------

【文本加解密】
  1. 切换到「文本加解密」标签页
  2. 输入密码（请牢记，解密时需要相同密码）
  3. 在「输入文本」框输入要加密的内容
  4. 点击「加密」按钮，密文会显示在输出框中
  5. 解密时，将密文粘贴到输入框，输入相同密码，点击「解密」

【文件加解密】
  1. 切换到「文件加解密」标签页
  2. 点击「浏览...」选择要加密的文件
  3. 输入密码，点击「加密」，会在同目录生成 .enc 加密文件
  4. 解密时，选择 .enc 文件，输入相同密码，点击「解密」

【MD5 计算】
  1. 切换到「MD5 计算」标签页
  2. 输入文本或选择文件，点击「计算 MD5」
  3. 可将两个 MD5 值粘贴到对比框中比对是否一致

四、注意事项
-----------
  - 密码一旦遗忘，加密数据无法恢复（AES-256 无后门）
  - 加密文件会生成 .enc 后缀的新文件，原文件不会被删除
  - 解密文件时，若密码错误会提示「密码错误或文件损坏」
  - 本工具为离线工具，所有运算在本地完成

五、技术信息
-----------
  - 加密算法：AES-256-GCM（认证加密）
  - 密钥派生：PBKDF2-SHA256（10万次迭代）
  - 编码方式：Base64
  - 构建方式：PyInstaller 单文件打包
