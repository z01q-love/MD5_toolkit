# MD5 + AES 加密解密工具 — 设计规格

## 概述

单文件 Python Tkinter 桌面工具，提供 AES-256 加解密 + MD5 哈希校验。
使用场景：加密文件 → 分享 → 对方解密，MD5 验证完整性。

## 架构

```
main.py          # 单文件，约300行
  ├─ AESEngine   # AES-256-GCM 加解密（pycryptodome）
  ├─ MD5Engine   # MD5 哈希计算（hashlib）
  └─ App(tk.Tk)  # GUI 界面
       ├─ Tab1: 文本加解密
       ├─ Tab2: 文件加解密
       └─ Tab3: MD5 计算/校验
```

## 功能

### 标签页1 — 文本加解密
- 输入文本 → 输入密码 → 点击加密 → 显示密文(Base64)
- 粘贴密文 → 输入密码 → 点击解密 → 显示原文
- 一键复制结果

### 标签页2 — 文件加解密
- 选择本地文件 → 输入密码 → 加密 → 生成 `文件名.enc`
- 选择 `.enc` 文件 → 输入密码 → 解密 → 生成 `文件名.dec`（恢复原名）
- 加密时自动显示 MD5 值，解密时自动校验

### 标签页3 — MD5 计算
- 输入文本 → 显示 MD5
- 选择文件 → 显示 MD5
- 两个输入框对比校验

## 技术选型

| 项目 | 选择 | 原因 |
|------|------|------|
| 加密算法 | AES-256-GCM | 认证加密，防篡改 |
| 密钥派生 | PBKDF2-HMAC-SHA256 | 密码 → 安全密钥 |
| MD5 | hashlib.md5 | 标准库，无需安装 |
| AES库 | pycryptodome | 纯 Python，跨平台 |
| GUI | tkinter | Python 自带，零依赖 |
| 打包 | 后期可选 PyInstaller | 当前 .py 直接运行 |

## 依赖

- Python 3.8+（标准库 tkinter + hashlib）
- pycryptodome（唯一第三方依赖）

## 错误处理

- 密码错误 → 明确提示"密码错误或文件损坏"
- 文件不存在 → 路径校验
- 空输入 → 按钮禁用或提示

## 测试

- 加密后解密还原验证
- 错误密码拒绝解密
- 篡改文件校验失败
- 中英文文本支持
